# faceless-shorts-agent
A local agent pipeline that generates short-form video packages:
ideas, scripts, captions, TTS voiceover, and visual plans.

Current MVP:
- Generate niche-based video ideas
- Score and select the best idea
- Write and edit a short-form script
- Generate AI voiceover
- Produce captions and visual beat sheet

Future:
- Trend research
- Stock footage search
- Video assembly
- YouTube upload scheduling
